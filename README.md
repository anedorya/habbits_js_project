# Трекер привычек (NestJS + PostgreSQL). Что можно сделать:
1. Есть "базовый" список из 10 привычек + можно добавить новую.
2. Можно "привязать" привычку к юзеру (пока только один дефолтовый юзер).
3. Подключить Google календарь и отправить себе напоминание о событии (нужны настройки).
4. Перестать отслеживать привычку.
5. Удалить созданную привычку (10 базовых привычек удалить нельзя).


### 1. Установка

в habbits_api run:
npm i
npm run start:dev

в habbits_web run:
npm i
npm run dev

Документация API лежит в habbits_api/README.md
